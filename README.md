# Semillero de Investigación — Identidad visual

Subcarpeta con el sistema visual propio del **Semillero de Investigación**: una identidad
universal para la investigación en todas sus ramas (no atada a una sola disciplina).

## Contenido
- `index.html` — guía de identidad visual completa: fundamento, logo, color, tipografía,
  tipos de publicación, plantillas de post y voz.
- `logo.html` — **marca del logo** (brand de la semilla): versiones, lockups, área de
  protección, tamaño mínimo, color y usos correctos/incorrectos.
- `assets/semillero.css` — tokens y estilos del sistema (independiente de la marca Daril Yovani).
- `assets/logo.svg` — logo maestro de la semilla simétrica que se ramifica.
- `assets/favicon.svg` — favicon (misma semilla).

## Resumen del sistema
- **Concepto:** sembrar preguntas, ramificar conocimiento.
- **Logo:** semilla → tallo → ramas con nodos (las disciplinas); el nodo que florece va en verde.
- **Color:** tinta `#16241F` + papel, con un único acento, verde brote `#1F9C7C` (≤10%).
- **Tipografía:** Fraunces (titulares) + Space Grotesk (sistema/datos) + Inter (cuerpo) + IBM Plex Mono (ficha).
- **Formatos:** Dato, Convocatoria, Reseña de paper, Explainer/Hilo, Video/Reel, Cita.

Abrir: `semillero/index.html` en el navegador.
